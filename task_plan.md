# Task Plan：客服智能知识助手（2026-03-06）

## 一、目标
将项目改造成客服智能知识助手，聚焦：知识问答、SOP 执行、结构化回复建议、升级决策与运营分析。

## 二、范围约束
1. 本轮不做 PPT 生成功能。
2. 上传支持两种方式：
   - 后台批量导入（主）
   - 前端上传管理页（后端中转）
3. FastGPT 以社区版能力为基线，商业版专属能力暂不纳入 MVP。
