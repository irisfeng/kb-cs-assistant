# 天翼视联绑定异常：网络与 2.4G 检查要点 v1

## 适用范围
适用于用户反馈摄像头无法绑定、扫码配网失败、绑定时提示网络异常等场景。

## 已核验依据
- 来源 1：`天翼视联-中国电信视联百川客户端客服文档-全国-20250530V4.0.docx`
- 已核验原文要点：
  - 视联百川二维码配网时需输入 WiFi 密码。
  - 设备仅支持 `2.4G` WiFi 网络。
  - 二维码需放在镜头前 `15-30cm`，直至提示“二维码扫描成功”。
  - 双频路由器场景下，需确认当前连接的是 `2.4G` WiFi。
- 来源 2：`天翼视联-中国电信行业版AI产品政企版客服文档-全国-20250427V3.0.docx`
- 已核验原文要点：
  - 绑定失败多与网络异常相关。
  - 设备联网并重启后，建议等待 `60 秒` 再继续后续操作。

## 使用边界
1. 当前已核验资料支持先做网络与 2.4G 前置检查。
2. 当前资料不支持把某个客户端的具体入口路径套用到所有产品。
3. 若用户尚未说明客户端、失败阶段和异常提示，应先分诊，不要直接给统一操作结论。

## 建议入库正文
```md
# 天翼视联绑定异常：网络与 2.4G 检查要点 v1

## 适用范围
适用于用户反馈摄像头无法绑定、扫码配网失败、绑定时提示网络异常等场景。

## 已知规则
1. 视联百川二维码配网时，设备仅支持 2.4G WiFi。
2. 双频路由器场景下，需先确认手机当前连接的是 2.4G WiFi。
3. 二维码需放在摄像头镜头前 15-30cm，直到提示“二维码扫描成功”。
4. 绑定失败多与网络异常相关；设备联网并重启后，建议等待 60 秒再继续后续操作。

## 标准回复口径
建议回复：您好，绑定异常先请您确认当前使用的是哪个客户端，并补充当前是否提示网络异常、是否使用 2.4G WiFi、二维码是否已在镜头前 15-30cm 范围内扫描成功。若设备已联网并重启，建议等待 60 秒后再继续尝试。  
处理步骤：先确认客户端和失败阶段；如属于扫码配网或网络异常场景，优先核 2.4G WiFi、二维码扫描距离、设备联网状态；如前置检查后仍失败，再判断是否需要人工复核。  
核验要点：客户端名称、失败阶段、网络提示、当前 WiFi 频段、设备联网状态、报错截图。  
升级建议：如完成网络和 2.4G 检查后仍无法绑定，建议补充截图或录屏后转人工复核。  
引用依据：视联百川配网需输入 WiFi 密码且设备仅支持 2.4G WiFi；二维码扫描距离建议 15-30cm；绑定失败多与网络异常相关，设备联网并重启后建议等待 60 秒。  

## 禁止直接承诺
1. 不能在未确认客户端前直接给统一绑定路径。
2. 不能在未确认 2.4G 与网络条件前直接判断设备故障。
3. 不能承诺“重试一次一定恢复成功”。
```
