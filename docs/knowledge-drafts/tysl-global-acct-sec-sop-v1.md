## 适用范围
适用于账号登录、账号识别、共享邀请、删除解绑确认、账号与绑定异常分诊场景。

## 核心原则
1. 先确认具体产品、APP 或客户端类型，再判断账号问题属于登录、共享、删除解绑，还是绑定异常。

4. 涉及高风险操作时，应先提醒用户确认影响范围，再决定是否转人工。

1. 先确认产品或客户端：视联百川 / 天翼云眼 / 其他。
2. 再确认问题类型：登录方式咨询 / 共享邀请 / 云回看删除 / 设备删除 / 套餐解绑 / 账号或绑定异常。

4. 最后判断是否属于常规可解释范围，还是需要人工录单支撑。

   - 短信验证码登录
   - 账号密码登录

建议回复：请先提供具体产品/客户端名称、账号标识和问题场景。当前账号类问题需要先区分是登录、共享、删除解绑，还是绑定异常后再判断处理方式。

核验要点：产品名称、手机号或登录账号、设备名称/设备 ID、问题发生时间、错误提示、是否涉及删除/解绑/共享邀请。  
升级建议：如涉及账号异常、绑定失败、异常登录、无法确认影响范围或需要录单支撑，建议转人工。
