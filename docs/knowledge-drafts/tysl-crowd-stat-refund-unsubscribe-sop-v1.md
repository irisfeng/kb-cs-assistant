## 来源映射
### 1. CRM 线下渠道
- 依据章节：`2.1.1 产品资费`、`2.1.2 订购模式及规则`、`3.1.1 CRM 退订规则`
- 提炼结论：
  - 产品指导价 20 元/月，各省可自行定价，实际资费以省内为准。
  - 线下 CRM 订购立即生效，订购当月功能费按日计扣，月底自动续订。
  - 退订立即生效，产品服务在月底终止，退订当月功能费全额收取。
  - CRM 自动续费月包可申请退订，退订次月生效。
  - CRM 非自动续费服务包一旦订购，不允许退订，到期自动退订。

### 2. 小翼管家第三方支付
- 依据章节：`2.2.1 产品资费`、`2.2.2 订购模式及规则`、`3.1.2`、`3.1.3`、`3.2.1`
- 提炼结论：
  - 小翼管家存在一次性月包/季包/年包，以及连续包月。
  - 一次性月包、季包、年包属于用户自主点击订购并付费，购买成功后不支持退订退费。
  - 连续包月可按退订流程取消，规则是“当月退订、次月生效”，退订后不再自动扣费。
  - 连续包月退订路径：
    - 支付宝：支付宝 -> 我的 -> 设置 -> 支付设置 -> 免密支付/自动扣款 -> 天翼数字生活 -> 关闭服务
    - 微信：微信 -> 我 -> 服务 -> 右上角菜单 -> 扣费服务 -> 天翼数字生活 -> 关闭服务
    - 翼支付：当前文档写明暂不支持

### 3. 退费争议处理
- 依据章节：`3.2.1 第三方支付订购退费流程`
- 提炼结论：
  - 一次性第三方支付订购，不支持退订退费，也不支持更换使用账号。
  - 连续包月第三方支付订购，不支持退费，但可按退订流程取消后续自动扣费。
  - 如已解释规则但用户仍坚持退费，且理由属于以下场景，可转人工支撑：
    - 业务故障导致无法使用
    - 套餐重复订购
    - 未使用过
  - 转人工前需至少收集：
    - 小翼管家 APP 登录号码
    - 第三方支付账单详情截图

### 4. 美好家旗舰店订单
- 依据章节：`2.3 美好家旗舰店订购`、`3.3 美好家旗舰店订单的退款/售后受理流程`
- 提炼结论：
  - 美好家旗舰店为单独线上订购入口。
  - 退款/售后路径：小翼管家 App -> 美好家旗舰店 -> 右上角个人中心 -> 退款/售后。
  - 申请通过后费用原路返回。
  - 具体退费政策以销售页面为准。

1. CRM 订购立即生效，订购当月功能费按日计扣，月底自动续订。

1. 第三方一次性月包、季包、年包：购买成功后不支持退订退费。
2. 第三方连续包月：不支持退费，但可以退订；当月退订、次月生效，退订后不再自动扣费。
3. 连续包月退订路径：

1. 美好家旗舰店订单退款/售后路径：小翼管家 App -> 美好家旗舰店 -> 右上角个人中心 -> 退款/售后。

3. 具体退费政策以销售页面为准。
