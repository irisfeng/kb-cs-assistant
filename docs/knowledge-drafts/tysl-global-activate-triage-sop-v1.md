## 使用边界
1. 当前已核验资料可以支持回答“激活/绑定类问题先怎么分诊、先补哪些信息、何时需要人工复核”。
2. 当前资料不支持在未确认客户端和失败阶段时，直接给统一绑定步骤。
3. 当前资料不支持承诺“无法绑定一定能立即恢复”，只能根据已知规则先排查并在必要时转人工。
4. “已被其他账号绑定”这类场景，可以说明“需原绑定用户先解绑”，但不能补写未引用明确给出的后台处理细节。
5. 行业版 AI 产品“无需单独配网”仅适用于该类产品在天翼云眼中的接入与套餐绑定，不应直接套用到所有摄像头产品。

# 天翼视联全局激活分诊 SOP v1

## 适用范围
适用于用户咨询“无法绑定”“无法激活”“摄像头接入失败”“绑定异常”，但尚未说明具体客户端、绑定阶段或错误提示的场景。

## 核心原则
1. 先区分当前是视联百川、天翼云眼还是其他产品/客户端场景，再判断具体绑定路径。
2. 先区分当前问题属于设备配网绑定、摄像头接入绑定，还是 AI 套餐绑定，不能混用规则。

## 标准分诊顺序
1. 先确认产品或客户端名称：视联百川 / 天翼云眼 / 其他。
2. 再确认失败阶段：配网绑定 / 设备接入 / 套餐绑定 / 其他。
3. 再确认异常提示：网络异常 / 已被其他账号绑定 / 无法扫码 / 无法定位 / 套餐无法绑定 / 其他。

建议回复：请先补充您使用的产品或客户端名称，以及当前是配网绑定、设备接入还是套餐绑定失败。不同客户端和场景的激活绑定规则不同。  
处理步骤：先确认客户端、失败阶段和报错提示，再判断是检查 2.4G 网络、二维码/UID 接入、原账号解绑，还是转人工复核。  
核验要点：产品或客户端名称、账号/手机号、设备名称、设备 ID/UID/CTEI、报错截图、问题发生时间。  
升级建议：如已重试仍无法绑定、提示已被其他账号绑定、网络检查后仍失败，建议转人工复核处理。  
引用依据：视联百川配网需要区分有线或二维码方式且设备仅支持 2.4G WiFi；天翼云眼支持二维码或 UID 方式接入，提示已被其他账号绑定时需原绑定用户先解绑。

2. 网络、重启、等待后仍无法绑定。
3. 提示已被其他账号绑定，但当前无法联系原绑定用户。
4. AI 套餐无法正常绑定到设备。
