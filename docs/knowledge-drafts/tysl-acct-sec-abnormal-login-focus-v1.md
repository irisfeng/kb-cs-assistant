# 天翼视联账号异常登录分诊口径 v1

## 适用范围
适用于用户反馈“账号异常登录”“账号疑似被盗”“想修改绑定信息”“账号安全异常”等场景。

## 已核验依据
- 来源 1：`天翼视联-中国电信视联百川客户端客服文档-全国-20250530V4.0.docx`
- 已核验原文要点：
  - 视联百川 APP 支持短信验证码登录、账号密码登录。
  - 共享邀请以视联百川 APP 账号即手机号为准。
  - 删除云回看视频需要验证码确认。
  - 删除设备后，如已绑定云回看或 AI 套餐，将全部自己解绑。
- 来源 2：`天翼视联-中国电信行业版AI产品政企版客服文档-全国-20250427V3.0.docx`
- 已核验原文要点：
  - 录单前应收集：天翼云眼登录账号、问题描述/故障现象、截图或录屏、设备标签截图、`UID/CTEI`、客户端版本号、手机品牌型号、联系电话。

## 使用边界
1. 当前已核验资料没有明确给出“忘记密码找回、账号被盗处置、异常登录拦截、账号冻结解封”的完整流程。
2. 因此这类问题可以做分诊和信息收集，但不能编造账号恢复、强制解绑、后台改绑等固定动作。
3. 当前更适合形成“保守但可引用”的安全口径，而不是假装已有完整处理流程。

# 天翼视联账号异常登录分诊口径 v1

## 适用范围
适用于用户反馈账号异常登录、账号疑似被盗、想修改绑定信息、账号安全异常等场景。

## 标准回复口径
建议回复：您好，账号异常登录或疑似被盗属于安全敏感场景。当前已知资料无法直接给出固定恢复流程，请您先提供产品或客户端名称、登录账号或手机号、异常提示、问题发生时间以及相关截图，我们先协助核清影响范围，再转人工复核。

核验要点：产品或客户端名称、手机号或登录账号、异常提示、问题发生时间、设备名称/UID/CTEI、截图或录屏。  
升级建议：如涉及异常登录、账号疑似被盗、绑定信息需要人工调整，建议转人工复核，不直接在线承诺处理结果。

## 禁止直接承诺
1. 不能直接承诺可在线找回密码。
2. 不能直接承诺客服可后台修改绑定信息。
3. 不能直接承诺异常登录一定如何恢复。
4. 不能在未确认影响范围前指导用户直接删除设备或解绑套餐。
