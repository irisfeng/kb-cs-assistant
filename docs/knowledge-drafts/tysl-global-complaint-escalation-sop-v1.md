## 使用边界
1. 本 SOP 解决的是“投诉/升级/人工介入怎么分诊”，不是“具体产品一定怎么赔付、多久办结”。
2. 具体退费、退订、退换货、售后、费用争议等口径，仍以对应产品文档、销售页面、省内政策和订单渠道规则为准。

# 天翼视联全局投诉升级 SOP v1

## 适用范围
适用于用户对处理结果不满意、要求投诉升级、要求人工介入、订购争议、费用争议、产品功能疑难问题等场景。

## 核心原则
1. 省内优先处理用户咨询、投诉、障碍申告，以及省内 CRM 销售品受理、变更、费用查询、费用争议、销售品内容解释、省内营销政策、订购退订、套餐绑定解绑等问题。
2. 天翼视联公司主要负责省 10000 号疑难支撑，以及产品功能使用类问题。
3. 投诉疑难问题可由天翼视联公司客服专席协同产品客服处理。
4. 如用户有其他订购疑问或需要投诉指引，可派集团单至“视联网公司”，或引导联系天翼视联客服处理。

## 标准处理顺序
1. 先确认产品名称、问题场景和用户当前诉求。
2. 再判断是省内 CRM/费用争议/订购退订/绑定解绑问题，还是产品功能使用问题。
3. 如属于一般咨询或常规解释范围，先按知识库规则答复。
4. 如用户明确要求投诉升级、对现有处理不满意、涉及疑难故障或多次未解决，转人工或派集团单进一步处理。

## 标准回复口径
建议回复：已记录您的投诉/升级诉求，请先提供产品名称、账号或订单/工单信息、问题描述和已处理情况，我们将根据具体产品和问题类型进一步复核。  
处理步骤：先确认产品、诉求和当前处理进展；再判断应由省内处理，还是转天翼视联公司客服专席/相关产品客服继续跟进。  
核验要点：产品名称、账号/联系电话、订单号或工单号、问题描述、发生时间、历史处理记录、相关截图或录屏。  
升级建议：如用户明确要求投诉升级、处理结果不满意、问题多次未解决，或属于功能使用疑难问题，建议转人工或派集团单至“视联网公司”。  
引用依据：省内负责投诉、障碍申告及 CRM/费用争议等问题；天翼视联公司负责疑难支撑和功能使用问题；全业务客服热线 400-9866-000，天翼云眼客服热线 400-9899-000。
