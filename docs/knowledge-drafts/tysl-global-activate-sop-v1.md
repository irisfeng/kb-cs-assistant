  - 网络问题是视联百川配网失败的高频前置检查项。

# 天翼视联全局激活绑定 SOP v1

## 适用范围
适用于设备配网、设备绑定接入、AI 套餐绑定、绑定失败分诊、激活绑定转人工场景。

## 核心原则
1. 先区分用户使用的是视联百川还是天翼云眼客户端，再判断对应的激活绑定路径。
2. 先区分当前问题属于“设备配网绑定”“摄像头接入绑定”还是“AI 套餐绑定”，不能混用规则。

1. 先确认产品或客户端：视联百川 / 天翼云眼 / 其他。
2. 再确认问题类型：设备配网绑定 / 摄像头绑定接入 / AI 套餐绑定 / 绑定失败。
3. 再确认当前报错或异常提示：网络异常 / 已被其他账号绑定 / 无法扫码 / 无法定位 / 套餐无法绑定。
4. 最后判断是按已知规则解释，还是转人工录单处理。

## 视联百川配网绑定规则
1. 视联百川可通过扫描设备机身二维码进行设备配网绑定。
2. 有线方式下，需先按设备说明书重置设备，确认前置操作完成并检查网线连接正常，再继续绑定。
3. 二维码方式下，需输入 WiFi 密码，设备仅支持 2.4G WiFi 网络。
4. 生成二维码后，应将手机屏幕二维码放在摄像头镜头前 15-30cm，直到提示“二维码扫描成功”。
5. 绑定成功后可查看实时视频，并可保存设备二维码。

## 天翼云眼绑定规则
1. 天翼云眼 APP 可通过首页右上角“添加设备”入口接入设备。
2. 可扫描设备二维码或手动输入设备 UID 完成绑定。
3. 绑定进度显示 100% 表示绑定成功。
4. 绑定成功后需继续进行设备定位，否则无法正常实时监控。

## AI 套餐绑定规则
1. 行业版 AI 产品不需要单独为 AI 套餐重新配网。
2. 在营业厅订购行业版 AI 标准应用及套餐后，可在天翼云眼 APP 内选择对应服务/套餐并绑定到设备。
3. AI 套餐绑定的前提是设备已在客户端内完成接入绑定。

## 绑定失败分诊规则
1. 若设备绑定失败，优先核查网络状态是否正常。
2. 若为双频路由器环境，先确认当前连接的是 2.4G WiFi。
3. 若提示设备已被其他账号绑定，需原绑定用户先解绑。
4. 若设备联网并重启后重新尝试，建议等待 60 秒再继续绑定流程。

处理步骤：先核产品/客户端、问题类型、异常提示；再按已知规则判断是检查 2.4G 网络、重新扫码/输入 UID、确认原账号解绑，还是转人工录单处理。  
核验要点：产品/客户端名称、手机号或登录账号、设备名称、设备 ID 或 UID/CTEI、异常提示、问题发生时间、截图或录屏。  
升级建议：如设备已被其他账号绑定、网络异常反复无结果、无法完成定位、AI 套餐无法正常绑定，建议转人工复核处理。  
引用依据：视联百川配网绑定支持扫码、有线和二维码方式，设备仅支持 2.4G WiFi；天翼云眼支持添加设备并通过 UID 接入，绑定 100% 表示成功；已被其他账号绑定时需原账号解绑。
